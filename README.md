# Indian Demography Dashboard

![Data Analysis](https://img.shields.io/badge/Data-Analysis-blue)
![Excel](https://img.shields.io/badge/Excel-Dashboard-green)

This repository contains data analysis resources and reports focused on the demography of India. The project involves raw census data on villages, towns, households, and population, transformed into analytical summaries and a comprehensive report.

## 📂 Project Structure

This project is primarily driven by data files and reports:

- `A-1_NO_OF_VILLAGES_TOWNS_HOUSEHOLDS_POPULATION_AND_AREA.xlsx`: The raw demographic dataset containing the number of villages, towns, households, population figures, and geographical area data.
- `savinay-working-file.xlsx`: The main working Excel file where the raw data is cleaned, processed, and transformed for analysis.
- `analysis-summary.xlsx`: The final analytical summary, likely containing the aggregated points, pivot tables, and dashboard components derived from the working file.
- `savinay-singh-report.docx`: A comprehensive written report documenting the findings, insights, and conclusions drawn from the demographic data analysis.

## 🚀 Usage

To explore the analysis:
1. Clone the repository:
   ```bash
   git clone https://github.com/savinaysingh7/Indian-Demography-Dashboard.git
   ```
2. Open the `analysis-summary.xlsx` file in Microsoft Excel or a compatible spreadsheet application to view the aggregated dashboard data.
3. Refer to `savinay-singh-report.docx` for the detailed narrative and insights of the demographic analysis.
